# Customer Purchase Analysis

## Description
This project analyzes customer purchase data to understand spending behavior using Python.

## Dataset
The dataset contains customer purchase records with:
- Customer Name
- Product
- Category
- Quantity
- Price

## Analysis Performed
- Loaded CSV data using Pandas
- Created a new feature: TotalAmount
- Calculated total spending per customer
- Visualized customer spending using a bar chart

## Tools Used
- Python
- Pandas
- Matplotlib
- Git & GitHub

## Key Insight
Riya is the highest spending customer based on total purchase amount.
